# Generated Exports

Generated app-ready preset files belong here.

Rules:

- Do not edit export files by hand.
- Keep export files deterministic so diffs are reviewable.
- Only reviewed authoring entries should be exported for app bundling.
- The iOS app should consume files from this directory, not draft authoring files.

Planned export targets:

- `jouyou-kanji-v1.json`
- `basic-vocabulary-v1.json`

Draft sample exports use separate filenames:

- `jouyou-kanji-v1.sample.json`
- `basic-vocabulary-v1.sample.json`
