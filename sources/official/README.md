# Official Sources

Track source metadata here before using source facts in app presets.

## Jouyou Kanji

- Source name: `常用漢字表`
- Publisher: Agency for Cultural Affairs, Government of Japan
- Publication form: Cabinet notification (`平成22年内閣告示第2号`)
- Scope: 2136 Jouyou Kanji, official readings, and official example words
- Copyright position: Japanese Copyright Act Article 13 excludes laws,
  notifications, instructions, circular notices, and similar government works
  from copyright protection.

## Usage Boundary

Allowed as source facts:

- Kanji characters
- Official readings
- Official example words
- Official table ordering and source labels

Not covered by this source:

- Korean meanings
- Flippers learning explanations
- New Japanese example sentences
- Korean example translations
- Vocabulary preset curation

Those must be authored by the project or sourced from a separately approved
license.
